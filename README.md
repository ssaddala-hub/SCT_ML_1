# House Price Prediction using Machine Learning

## Objective
Build a machine learning model to predict house prices based on various features such as area, bedrooms, bathrooms, and other property characteristics.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Algorithm
Linear Regression

## Steps Performed
1. Data loading and preprocessing
2. Feature selection
3. Train-test split
4. Model training using Linear Regression
5. Prediction and evaluation

## Outcome
Developed a machine learning model capable of predicting house prices based on input features.
